🍽️ La Bella Cucina — Italian Restaurant Website

A modern, fully responsive restaurant website UI built using HTML5, CSS3, and Vanilla JavaScript.

This project focuses on elegant UI design, smooth animations, and a premium restaurant experience — built without any frameworks or external dependencies.
